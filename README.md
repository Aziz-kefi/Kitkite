# Kitkite
kitKite is an application for kitesurfing lovers and offers features for users like event , kiting schools , kiting certification points , kiting spots etc..


# BackEnd:
 Developped with: 
 Node.js (JavaScript)
 Express.
 DataBase and DataBase Managment:
 MongoDB.
 Mongoose.
 Implemented tools : 
 SwaggerUI
 file.rest
